# Segmentación y visualización de datos CFD para proceso DED

Proyecto de apoyo técnico desarrollado para segmentar y visualizar datos asociados a simulaciones CFD del proceso de deposición directa de energía (DED).

## Objetivo

Filtrar datos numéricos provenientes de archivos de simulación CFD y generar visualizaciones 2D de los datos segmentados para facilitar su análisis.

## Herramientas utilizadas

- Python
- Pandas
- NumPy
- Matplotlib

## Actividades realizadas

- Lectura de archivos de datos `.SPY`.
- Reorganización de matrices de datos.
- Segmentación de datos por intervalo.
- Exportación de datos segmentados.
- Visualización 2D mediante gráficas de dispersión.
